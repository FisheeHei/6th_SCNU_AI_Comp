# 方案采用思路  
对数据集建立TF-IDF向量器并进行分词  

将训练集的数据按照难度进行分类，选作各难度的“题库”  

对测试集中的每条数据，都进行和同难度“题库”全部SQL模式的相似度计算（这里采用余弦相似度）     

并将相似度前五的模式收纳入基线的prompt成为带有参考的“强化prompt”中交予LLM以强化推理准确度    

此外对基线模型控制参数进行了调整。
